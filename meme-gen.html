Meme Generator

This Meme Generator is used to add text to any selected image.

How does it work:
You choose a file by clicking on Choose file button, which takes you to a folder you can select any image from.
To add text to the image, you use the Top Text and Bottom Text text editors.
To increase the size of your text using the sliders labelled Text size.
To draw on your image, you use the Draw button.
To remove the lines added to your image, you use the Erase button.
